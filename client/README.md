## Overview
This project is being built with a "Modern-Minimal" aesthetic, focusing on:
- **Soft-Glass Visuals**: Multi-layered depth and blur.
- **Electric Blue**: High-voltage accent color.
- **Typography**: Outfit (Headings) + Manrope (Body).

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **State/Theme**: `next-themes`
- **Validation**: `zod`
- **Animation**: `framer-motion`

## Getting Started

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Run development server**:
   ```bash
   pnpm dev
   ```

3. **Build for production**:
   ```bash
   pnpm build
   ```
